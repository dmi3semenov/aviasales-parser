## ✅ Сделано (апрель 2026)

- `trip-config.json` — единственный источник параметров (маршрут, окна вылета, ночи, бюджет, критерии)
- `generate-trips.js` — читает конфиг, поддерживает несколько окон вылета, любой маршрут, флаг `--run`
- `aviasales-parser.js` — критерии из конфига, лист "По датам", "Все варианты", фильтр цен 5k–300k ₽, `viewport: null` (браузер не скошен)
- `build-summary.js` — полноценная сводка на 10 вкладок:
  - **Итог** (оранжевая) — берём / чуть дорого / не берём по абсолютному порогу
  - **По времени вылета** (жёлтая) — 2D-сетка слотов с лучший→/↓
  - **Рек - Карточки / Таблица / По дате / По окнам** (зелёные)
  - По датам, Все варианты, Все сводки, Пояснения
- Цветные вкладки (жёлтая/зелёная/оранжевая), правильный порядок
- Столбец "Кол-во" (количество вариантов рейсов) в таблицах
- Имя файла содержит маршрут: `as_СВОДКИ_Москва-Минск_...xlsx`
- Дедупликация: берётся только последний файл на каждую комбинацию дат

## Приоритет 1 — следующий маршрут

### Как поменять маршрут
1. Открыть `trip-config.json`
2. Поменять `"города"`, `"названия"`, `"окна_вылета"`, `"ночи"`, `"бюджет_макс"`
3. `node generate-trips.js` — проверить комбинации
4. `node generate-trips.js --run` — запустить парсинг, решить капчу
5. `node build-summary.js` — собрать сводку
